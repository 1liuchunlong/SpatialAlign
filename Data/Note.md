# Dataset Access

SpatialAlign example data are not included in the repository due to size. Download them externally and place the files in this directory.

---

### Required Files

| Filename | Description |
|----------|-------------|
| `sc_nsclc.h5ad` | scRNA-seq reference data (NSCLC) |
| `lung9_rep1_15types.h5ad` | Spatial transcriptomics target data (15 cell types) |
| `X_umap_df.csv` | Pre-computed UMAP coordinates for visualization |

### Download

> [Download SpatialAlign Example Datasets](https://drive.google.com/drive/folders/1ybNLg5KB2sEc2eue2FP2BrOQtMi1FrIo?usp=drive_link) (Google Drive)

After downloading, place the files in `Data/` and update the paths in `demo-nsclc.ipynb` to run the demo.
